# Escape From Gravity

2021-1 학기 SW설계기초 C언어를 이용한 4인 협업 게임 프로젝트.


C언어의 기본적인 기능만 사용해 제작했습니다. 게임은 충돌 처리를 통해 캐릭터가 장애물을 피하며 조건을 충족해 목적지에 도달하는 방식입니다.
